## About

A basic Remix app using TinaCMS for the content.

Set up Remix using the [Remix Quick Start](https://remix.run/docs/en/main/start/quickstart) guide and added TinaCMS CLI from the [Remix + Tina Setup](https://tina.io/docs/frameworks/remix/) guide.

Following the [TinaCMS Deep Dive](https://www.youtube.com/watch?v=PcgnJDILv4w) for the Tina setup.
